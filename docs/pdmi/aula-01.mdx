## 💻 Fundamentos de TypeScript

O TypeScript é a base do desenvolvimento seguro em React Native. Aqui estão os conceitos fundamentais que usaremos:

### 1. Const e Let — Variáveis e Constantes

```typescript
const pi = 3.14;           // IMUTÁVEL (equivalente ao val)
let count = 0;             // MUTÁVEL (equivalente ao var)
count = 1;                 // Permitido
```

### 2. Tipagem Forte e Tipos Primitivos

```typescript
const nome: string = "Yuri";
const idade: number = 28;
const isDev: boolean = true;

// Inferência: TS deduz o tipo automaticamente
let cidade = "São Paulo"; // TS sabe que é string

// Union Types
let telefone: string | null = null;

// ⚠️ EVITE 'any' — desativa a verificação do TS
let dinamica: any = "Texto";
dinamica = 100; // O TS não avisa o erro!
```

### 3. Null Safety

```typescript
const bio = usuario.profile?.bio;           // Optional Chaining
const github = user.github ?? "Não Cadastrado"; // Nullish Coalescing
```

### 4. Arrow Functions

```typescript
const somar = (a: number, b: number) => a + b;
const saudar = (nome: string, msg: string = "Olá") => `${msg}, ${nome}`;
```

### 5. Arrays e Pipeline Funcional

```typescript
const techs: string[] = ["React Native", "Expo", "TypeScript"];
techs
    .filter(t => t.length > 4)
    .map(t => t.toUpperCase())
    .forEach(t => console.log(t));
```

### 6. Interfaces

```typescript
interface User {
    id: string;
    name: string;
    email: string;
    bio?: string; // Opcional
}
```

### 7. Destructuring e Spread

```typescript
const { name, email } = meuPerfil;
const novo = { ...meuPerfil, name: "Outro" };
```

### 8. Classes

```typescript
class Aluno {
    constructor(public nome: string, public nota: number) {}
    apresentar() { console.log(`Olá, sou ${this.nome}`); }
}
```

### 9. Import e Export

```typescript
export const API_URL = "https://api.merge.com";
import { API_URL } from "./config";
```

## 🎨 Estilização e Customização

### StyleSheet.create
Sempre use `StyleSheet.create({})` ao invés de objetos inline — é mais performático e o editor sugere propriedades.

### Flexbox no React Native

```tsx
// Direção padrão: column (vertical)
<View style={{ flexDirection: 'row' }}>  {/* Muda para horizontal */}
    <View style={{ flex: 1 }} />  {/* Ocupa 50% */}
    <View style={{ flex: 1 }} />  {/* Ocupa 50% */}
</View>
```

| Propriedade | O que faz |
|:------------|:----------|
| `flex: 1` | Ocupa todo o espaço disponível |
| `flexDirection` | `'column'` (vertical) ou `'row'` (horizontal) |
| `alignItems` | Alinha no eixo cruzado (`center`, `flex-start`, `flex-end`) |
| `justifyContent` | Alinha no eixo principal (`center`, `space-between`) |
| `gap` | Espaçamento entre itens |

### Composição de Estilos

```tsx
{/* Combina dois objetos de estilo usando array */}
<View style={[styles.card, styles.cardDestaque]} />

{/* Estilo condicional */}
<TouchableOpacity style={[styles.button, loading && styles.buttonDisabled]} />
```

### Sombras
```tsx
const styles = StyleSheet.create({
    card: {
        // iOS
        shadowColor: '#000',
        shadowOffset: { width: 0, height: 4 },
        shadowOpacity: 0.1,
        shadowRadius: 8,
        // Android
        elevation: 4,
    },
});
```

## ⚡ useEffect — Efeitos Colaterais

```tsx title="app/index.tsx"
import { useEffect } from 'react';

useEffect(() => {
    // Este código roda APÓS o componente montar na tela
    const timer = setTimeout(() => {
        router.replace('/login');
    }, 2000);

    // Cleanup: roda quando o componente DESMONTA
    return () => clearTimeout(timer);
}, []); // [] = roda apenas 1 vez
```

| Array de Dependências | Comportamento |
|:----------------------|:-------------|
| `[]` (vazio) | Roda **1 vez** ao montar |
| `[variavel]` | Roda quando `variavel` muda |
| sem array | Roda a **cada re-render** (⚠️ cuidado!) |

## 🧠 useState — Gerenciamento de Estado

```tsx title="app/login.tsx"
import { useState } from 'react';

const [email, setEmail] = useState('');       // Começa vazio
const [loading, setLoading] = useState(false); // Começa false
```

| Conceito | Exemplo |
|:---------|:--------|
| Ler o valor | `email` |
| Atualizar o valor | `setEmail('novo@email.com')` |
| Nunca faça isso | ~~`email = 'novo@email.com'`~~ |

### TextInput com useState

```tsx
<TextInput
    value={email}                    // Exibe o valor atual
    onChangeText={setEmail}          // Atualiza a cada tecla
    keyboardType="email-address"     // Teclado de email
    secureTextEntry                  // Esconde o texto (senhas)
/>
```

### Login com Formulário e Estado

```tsx title="app/login.tsx"
async function handleLogin() {
    setLoading(true);
    try {
        await apiClient.auth.signIn(email, password);
        router.replace('/register');

    } finally {
        setLoading(false); // Roda SEMPRE
    }
}
```
