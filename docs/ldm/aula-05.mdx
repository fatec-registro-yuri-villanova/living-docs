# Aula 05: Ktor Server Architecture

:::info 🔀 Branch da Aula
Esta aula refere-se à branch: **`aula-05`**
```bash
git checkout aula-05
```
:::

## 📖 Conceito

À medida que nossa API cresce, configurar tudo diretamente dentro do `Application.kt` torna o código ilegível. O **Ktor** suporta a abstração por **Plugins**. Nesta aula, extraímos responsabilidades cruciais (como CORS, Tratamento de Erros, JSON Parsing e Roteamento) para funções extensórias dedicadas alojadas na pasta `plugins/`.

## 💻 Código de Referência

Abaixo estão os exemplos de plugins construídos:

### 1. Serialization Plugin

```kotlin title="server/src/main/kotlin/com/fatec/merge_skills_kmp/plugins/Serialization.kt"
package com.fatec.merge_skills_kmp.plugins

import io.ktor.serialization.kotlinx.json.*
import io.ktor.server.application.*
import io.ktor.server.plugins.contentnegotiation.*
import kotlinx.serialization.json.Json

fun Application.configureSerialization() {
    install(ContentNegotiation) {
        json(Json {
            prettyPrint = true
            isLenient = true
            ignoreUnknownKeys = true
        })
    }
}
```

### 2. StatusPages Plugin

```kotlin title="server/src/main/kotlin/com/fatec/merge_skills_kmp/plugins/StatusPages.kt"
package com.fatec.merge_skills_kmp.plugins

import io.ktor.http.*
import io.ktor.server.application.*
import io.ktor.server.plugins.statuspages.*
import io.ktor.server.response.*

fun Application.configureStatusPages() {
    install(StatusPages) {
        exception<Throwable> { call, cause ->
            call.respondText(text = "500: $cause", status = HttpStatusCode.InternalServerError)
        }
    }
}
```

### 3. Ajustando o Ponto de Entrada

Com os plugins isolados, nosso arquivo principal de servidor simplifica imensamente sua inicialização:

```kotlin title="server/src/main/kotlin/com/fatec/merge_skills_kmp/Application.kt"
fun Application.module() {
    // Configurações do Supabase (tentativa 1: JVM args / Fallback: local.properties)
    var supabaseUrl = System.getProperty("SUPABASE_URL") ?: System.getenv("SUPABASE_URL")
    var supabaseKey = System.getProperty("SUPABASE_KEY") ?: System.getenv("SUPABASE_KEY")

    if (supabaseUrl == null || supabaseKey == null) {
        val properties = java.util.Properties()
        val file = java.io.File("local.properties") // Busca recursiva omitida para brevidade
        if (file.exists()) {
            properties.load(java.io.FileInputStream(file))
            supabaseUrl = properties.getProperty("SUPABASE_URL")
            supabaseKey = properties.getProperty("SUPABASE_KEY")
        }
    }
    
    val supabase = if (supabaseUrl != null && supabaseKey != null) 
        createAppSupabaseClient(supabaseUrl, supabaseKey) else null

    // Pipeline configuration via Modular Plugins (Aula 05)
    configureSerialization()
    configureCORS()
    configureStatusPages()
    configureRouting(supabase)
}
```
